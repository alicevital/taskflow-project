Projeto: “TaskFlow” – API de Gerenciamento de Tarefas

Um sistema básico de gerenciamento de tarefas pessoais, ideal para treinar rotas, modelos, autenticação e integração com banco de dados.


Uma API que permite:

- Criar usuários e autenticar com token.
- Criar, listar, atualizar e excluir tarefas.
- Filtrar tarefas por status (pendente, em andamento, concluída).
